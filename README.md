# Playlist Manager Using Linked List

## Description
This project is a simple **playlist manager** implemented in Java using **linked lists**.  
It allows the user to **add songs**, **delete songs**, **display the playlist**, and **play the next song** in order.

This project demonstrates basic **Object-Oriented Programming (OOP)** concepts and **linked list operations** in Java.

---

## Features
- Add a song to the playlist
- Delete a song by name
- Display all songs in the playlist
- Play the next song in order

---

## How to Run

1. **Open terminal/command prompt** in the project folder.  
2. **Compile the Java files**:
   javac PlaylistApp.java
